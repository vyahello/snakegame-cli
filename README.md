# Terminal snake game
> A simple snake game right in your terminal using python built-in `curses` package.

**Tools/features**
 - python 3.7
 - curses
 - pytest
 - shell
 - travis/github CI

## Usage
Run script from the root directory of the project:
```bash
~ python game.py
```
